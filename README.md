# git-cheat-sheet
### Most used
Checkout branch
`git checkout branchname`
Pull & update local branch
`git pull`
Fetch changes (Note: this doesn't pull or update local branches)
`git fetch --all`

Stage changes
`git add --all` or `git add .`
Commit changes
`git commit -m "my commit message"`
Stage and commit
`got commit -a -m "my commit message"`
